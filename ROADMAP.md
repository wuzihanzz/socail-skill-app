## 📋 数据存储说明

### 用户数据存储位置
你的所有数据都存储在**浏览器的localStorage中**，具体包括：

```
浏览器 LocalStorage
└── "social-skill-game-state" (主要数据)
    ├── currentCharacterId      // 当前聊天的角色ID
    ├── relationships           // 每个角色的关系数据
    │   ├── chen-wei
    │   │   ├── trustLevel      // 信任度 (0-100)
    │   │   ├── satisfactionLevel // 满意度 (0-100)
    │   │   ├── currentEmotion  // 当前表情 (neutral/happy/upset)
    │   │   ├── unlockedSkills  // 已解锁的技能IDs
    │   │   ├── conversationHistory // 完整对话记录
    │   │   ├── askedAbout      // 是否问过名字/年龄等
    │   │   └── lastDailyEvent  // 最后一次事件日期
    │   ├── lin-xue
    │   └── xiao-mei
    └── conversationHistory     // 当前对话历史
```

### 为什么选择localStorage？
- ✅ **零服务器成本** - 数据存在你的电脑上
- ✅ **完全隐私** - 没有人能看到你的对话
- ✅ **离线可用** - 加载过的内容，断网也能看
- ✅ **自动保存** - 关闭网站后，下次打开进度完整保留
- ✅ **无需注册** - 打开即用，无账户系统

### 浏览器控制台查看数据
如果想看到你的所有数据，打开浏览器控制台（F12）：
```javascript
// 查看所有数据
console.log(JSON.parse(localStorage.getItem('social-skill-game-state')))

// 清空所有数据（重新开始）
localStorage.clear()
```

### 数据持久化原理
- 使用 **Zustand** 状态管理库
- 每次状态变化时，自动调用 `saveToStorage()` 保存到localStorage
- 应用启动时，从localStorage恢复之前的状态
- 详见：`src/store/gameStore.ts`
